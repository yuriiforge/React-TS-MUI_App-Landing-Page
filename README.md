# React + TypeScript + MUI Responsive App

This is a simple project demonstrating how to build a responsive React application using **TypeScript** and **Material-UI (MUI)**. The app adapts its layout depending on the screen size (desktop, tablet, mobile) using MUIâ€™s **`useMediaQuery`** and **Grid/Tabs** components.

## Features

- Built with **React** and **TypeScript**
- Responsive design with **MUI breakpoints**
- Example of **AppBar** and **Drawer** for mobile navigation
- Tabs that work on desktop
- Buttons and icons styled with MUI

## Getting Started

1. Install dependencies

```bash
npm install
```

2. Run the project

```bash
npm run dev
```
